# react-demos

## Features
- Sass
- webpack 4.x
- Create-react-app
- Redux + react-redux
- React Router v4
- px2rem

## Getting Started
Checkout this repo, install dependencies, then start the process with the following:

```bash
> git clone https://github.com/wobushijincan/react_demos.git
> cd react-demos
> npm install
> npm start
```

## Build the app
```bash
npm run build
```
